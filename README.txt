# MédiaSmart

MédiaSmart est une application de gestion de médiathèque développée par une équipe de 4 développeurs. Ce projet a été réalisé en 4 semaines.

## Fonctionnalités

### Côté utilisateur :
- Inscription et connexion
- Consultation du catalogue de livres
- Réservation de livres

### Côté bibliothécaire :
- Ajout, modification et suppression de livres
- Gestion des utilisateurs
- Gestion des emprunts et des retours
- Consultation de l'historique d'utilisation

## Technologies utilisées

- **Back-end** : PHP (POO, MVC)
- **Front-end**: HTML, CSS (SASS), JavaScript (vanilla + GSAP)
- **Base de données**: MySQL
- **Outils**: Figma, Git, Trello pour la gestion du projet

## Installation

1. Clonez le dépôt :
    ```sh
    git clone https://github.com/votre-repo/mediasmart.git
    ```

2. Installez les dépendances avec Composer :
    ```sh
    composer install
    ```
3. Importez le fichier SQL fourni dans votre base de données MySQL.

4. Lancez le serveur local :
    ```sh
    php -S localhost:8000
    ```

5. Accédez à l'application via `http://localhost:8000`.
